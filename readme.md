# My Profile Chatbot

A chatbot about my profile, experience, education and skills

## Project Structure

- basic.py: 
    - main workflow of the chatbot in cli
    - run with `python basic.py`
- ui_qa.py
    - streamlit app for QA chatbot
    - run with `streamlit run ui_qa.py`
